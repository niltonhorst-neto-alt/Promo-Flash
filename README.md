# Promo-Flash

## Descrição do Projeto

O Promo-Flash é um sistema de e-commerce especializado em Flash Sales para produtos eletrônicos. O projeto tem como objetivo oferecer uma plataforma capaz de suportar grandes volumes de acessos simultâneos durante promoções relâmpago, garantindo estabilidade, rapidez e segurança nas operações de compra.

A proposta surgiu devido aos problemas enfrentados por lojas virtuais tradicionais durante períodos de alta demanda, como lentidão no carregamento das páginas, falhas no checkout e inconsistências no controle de estoque. Esses problemas impactam diretamente a experiência do usuário e podem causar perdas financeiras para as empresas.

O sistema será desenvolvido para atender empresas do setor de comércio eletrônico que trabalham com campanhas promocionais de curta duração e grande fluxo de clientes.

## Problema Identificado

Atualmente, muitos sistemas de e-commerce apresentam dificuldades em lidar com picos de acesso simultâneo durante promoções relâmpago. Entre os principais problemas identificados estão:

* Lentidão no carregamento das páginas;
* Quedas do sistema durante períodos de alta demanda;
* Falhas no processo de checkout;
* Problemas de sincronização de estoque;
* Risco de vendas duplicadas de produtos;
* Baixa escalabilidade da infraestrutura.

O projeto busca solucionar esses problemas utilizando tecnologias modernas de desenvolvimento web e arquitetura escalável.

## Objetivos do Projeto

* Desenvolver uma plataforma de Flash Sales eficiente e escalável;
* Garantir estabilidade mesmo com muitos acessos simultâneos;
* Melhorar a experiência do usuário durante as compras;
* Implementar um sistema seguro de autenticação e pagamentos;
* Criar um controle de estoque confiável em tempo real;
* Aplicar boas práticas de desenvolvimento ágil utilizando Scrum.
